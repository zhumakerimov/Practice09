# Practice 7 - Pygame Games Collection

Three interactive games built with Pygame.

## Projects

### 1. Mickey's Clock
Analog clock that displays current system time using Mickey Mouse hands as clock hands.
- Right hand shows minutes
- Left hand shows seconds  
- Updates in real-time every second

**Controls:** Close window to exit

### 2. Music Player
Keyboard-controlled music player with playlist management.

| Key | Action |
|-----|--------|
| P | Play current track |
| S | Stop playback |
| N | Next track |
| B | Previous track |
| Q | Quit player |

### 3. Moving Ball Game
Control a red ball using arrow keys. Ball cannot leave the screen boundaries.

| Key | Action |
|-----|--------|
| ↑ | Move up 20px |
| ↓ | Move down 20px |
| ← | Move left 20px |
| → | Move right 20px |
